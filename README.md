# 스마트팩토리 설비 예지보전 데이터 분석 프로젝트

## 프로젝트 소개

본 프로젝트는 AI4I 2020 Predictive Maintenance Dataset을 활용하여 제조 설비 데이터를 분석하고,
설비 고장에 영향을 미치는 주요 요인을 파악하기 위해 수행한 데이터 분석 프로젝트입니다.

Python을 활용한 데이터 전처리 및 탐색적 데이터 분석(EDA)을 수행하였으며,
분석 결과를 바탕으로 스마트팩토리(MES) 환경에서 활용 가능한 개선 방안을 도출하였습니다.

---

## 프로젝트 목적

- 제조 설비 데이터의 특성을 파악한다.
- 설비 고장과 관련성이 높은 변수를 분석한다.
- 분석 결과를 기반으로 예방보전(Predictive Maintenance) 관점의 개선 방안을 제시한다.

---

## 개발 환경

- Python
- Pandas
- Matplotlib
- Seaborn
- PyCharm

---

## 데이터 소개

- AI4I 2020 Predictive Maintenance Dataset
- 데이터 개수 : 10,000건
- 변수 개수 : 14개
- 결측치 : 없음

주요 변수

- 제품 유형(L / M / H)
- 외기온도
- 공정온도
- 회전속도(RPM)
- 토크(Nm)
- 공구마모시간
- 설비고장 여부

---

## 분석 과정

### 1. 데이터 전처리

- 컬럼명 한글 변경
- 결측치 확인
- 기초 통계량 확인

### 2. 탐색적 데이터 분석(EDA)

- 제품 유형별 고장률 분석
- 설비고장 여부별 평균 토크 분석
- 설비고장 여부별 평균 회전속도 분석

### 3. 상관관계 분석

- Correlation Heatmap
- Box Plot
- Scatter Plot

---

## 주요 분석 결과

### 제품 유형별 고장률

제품 유형별 고장률을 비교한 결과 L 제품에서 가장 높은 고장률이 나타났다.

---

### 설비고장 여부별 평균 토크

고장 설비의 평균 토크가 정상 설비보다 높게 나타났다.

토크는 설비 고장과 가장 관련성이 높은 변수 중 하나로 확인되었다.

---

### 설비고장 여부별 평균 회전속도

회전속도는 설비 고장과의 관계가 토크보다 상대적으로 낮게 나타났다.

---

### 상관관계 분석

Heatmap 분석 결과 토크가 설비고장과 가장 높은 상관관계를 보였다.

또한 회전속도와 토크는 강한 음의 상관관계를 나타냈다.

---

## 결론

제품 유형 자체보다 설비 운전 조건이 설비 고장에 더 큰 영향을 미치는 것으로 확인되었다.

특히 토크가 설비 고장과 가장 관련성이 높은 변수로 나타났으며,
MES 환경에서는 토크를 실시간 모니터링하여 예방보전에 활용할 수 있을 것으로 판단된다.
